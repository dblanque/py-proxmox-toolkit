# Proxmox Python Toolkit

## Installation

```bash
$ apt update -y
$ apt install git python3 python3-venv -y
$ cd /opt/
$ git clone https://github.com/dblanque/py-pve-toolkit
$ cd ./py-pve-toolkit
$ bash setup_venv ./
```
